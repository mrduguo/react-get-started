# react-get-started

## Dev Build

    cd ~/github/mrduguo/react-get-started
    ./gradlew run
    

## All In One Build

    cd ~/github/mrduguo/react-get-started
    ./gradlew && git add -A docs && git commit -a -m "updates" && git push
